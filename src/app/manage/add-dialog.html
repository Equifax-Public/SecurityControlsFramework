Description

Copy your Control and TR files into the assets folder of the application. Follow naming pattern of "roleName - TR/Control.csv"

File requirements

Files must be in CSV format. A file submission for both Control and TR views is required in the assets folder.

TR View - Required Columns

Domain Name, Domain, TR_ID, Sub_TR_Order, Sub_TR_Indent, Sub_TR_ID, TR_Description, TR_Scope

Control View - Required Columns

Domain Name, Domain, Control_ID, Control_Desc, Control_Scope, Control_Owner, Control_Operator, Control_Type, Control_Type
